# 基础语法

### 语句
- 分辨一条语句的标准是已“;”结尾。一条语句可以分别在多行，多行语句也可以在一行。  
- SAS 关键字不区分大小写。  
- 必须使用一个运行语句结束。

### 变量
变量表示SAS数据集中的一列。
- 不超过 32 个字符。
- 字母、数字或“_”组成。
- 字母或下划线开头。

### 数据集
- DATA语句之后的一个字表示一个临时数据集的名字。这意味着数据集被在会话结束时删除。
- 数据集名称可以使用库名称作为前缀，这使它成为永久数据集。 这意味着数据集在会话结束后仍然存在。
- 如果省略SAS数据集名称，则SAS会创建一个临时数据集，其中包含由SAS生成的名称，例如 - DATA1，DATA2等。

### 注释

- 多行注释【*;】。  
    可以换行，但不能出现“;”或不匹配的引号。  

    ```sas
    * 单行注释 ;

    * 多行注释第一行
    * 多行注释第二行 ;
    ```
- 多行注释【/**/】。  
    可以换行，不能嵌套。  

    ```sas
    /* 单行注释 */

    /* 多行注释第一行
     * 多行注释第二行 */
    ```

> 【*;】注释的优点在于，作为单行注释时，可以删除结尾的“;”，起到同时注释下一条语句的作用。如：

```sas
*程序 a
data a;

*程序 b;
data b;

run;
```
